# ❓ - Kurulum Nasıl Yapılır?

1. .zip dosyasını indirin.
2. `install.bat` dosyasına tıklayın ve modülleri kurun.
3. `config.json` dosyasını açın
4. Hesabınızın token’ını girin.
5. Dosyayı kaydedin, ardından `run.bat` dosyasını çalıştırın.
6. Tarama tamamlanana kadar bekleyin.
7. Tarama süreci bitince, sonuçlar `scraped` klasöründe sunucunun **ID’si** ile kaydedilmiş olacak.
